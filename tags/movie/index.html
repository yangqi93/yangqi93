《三峡好人》观后感

2023-04-28 ::
#movie 
影片正式海报 2006 电影上映的那一年，我刚上初中。今天看的时候，低分辨率让我很不适应，十几年来分辨率快速发展，市面上已经能看到4k的电影了。 开头的群像让我以为在看纪录片，要不是实在没有好片可看了，我差点关掉网页。那个时候的特效真的只有五毛，剧情也不够跌宕起伏，节奏慢得像是停留在某一个时空，要不是听说这部片子特牛，怕是没有耐心看下去。 故事的主线是为了修建三峡水利工程，原有的土地会被淹没，需要迁走大量移民。其中一个有着两千多年历史的城市，要在两年拆完，这就是奉节老县城。镜头在这里展开，到处都是废墟和等着成为废墟的空楼，两个毫不相干的故事主人公因这些废墟产生了某种联系。韩三明来奉节寻找他的前妻和女儿，沈红来奉节寻找他两年没回家的丈夫，赶在三期蓄水前，这座城市还没有消失的时候。 很多描述小人物的作品都特别出彩，像宁浩的疯狂系列，周星驰的无厘头系列，他们满足了我们的想象，想象着总有一天咸鱼翻身，又或是走上人生巅峰，又或是家庭圆满。三峡好人没有满足我们的想象，它的镜头那么平静，也不会用俯视的视角描述生活的失落，我看到的就是我们的父辈，面对失落时的艰辛探索。展示的就是自称社畜的你我，展示的是真实的中国。 就像豆瓣影评说的，我们期望一部好电影，期望一部值得尊敬的电影，一个值得尊敬的导演。谢谢贾樟柯。
