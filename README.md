使用 Apidoc 自动生成笔记
==========================

[yii2-apidoc github address](https://github.com/yiisoft/yii2-apidoc)

安装
---------
- `git clone git@github.com:yangqiphp/note.git folder`
- `composer install` 需提前安装 [composer](http://www.phpcomposer.com/)

文档生成步骤
---------
README.md 将生成左侧导航栏
index.md 将生成索引
执行 doc.sh
