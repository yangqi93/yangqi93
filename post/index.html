《三峡好人》观后感

2023-04-28 ::
#movie 
影片正式海报 2006 电影上映的那一年，我刚上初中。今天看的时候，低分辨率让我很不适应，十几年来分辨率快速发展，市面上已经能看到4k的电影了。 开头的群像让我以为在看纪录片，要不是实在没有好片可看了，我差点关掉网页。那个时候的特效真的只有五毛，剧情也不够跌宕起伏，节奏慢得像是停留在某一个时空，要不是听说这部片子特牛，怕是没有耐心看下去。 故事的主线是为了修建三峡水利工程，原有的土地会被淹没，需要迁走大量移民。其中一个有着两千多年历史的城市，要在两年拆完，这就是奉节老县城。镜头在这里展开，到处都是废墟和等着成为废墟的空楼，两个毫不相干的故事主人公因这些废墟产生了某种联系。韩三明来奉节寻找他的前妻和女儿，沈红来奉节寻找他两年没回家的丈夫，赶在三期蓄水前，这座城市还没有消失的时候。 很多描述小人物的作品都特别出彩，像宁浩的疯狂系列，周星驰的无厘头系列，他们满足了我们的想象，想象着总有一天咸鱼翻身，又或是走上人生巅峰，又或是家庭圆满。三峡好人没有满足我们的想象，它的镜头那么平静，也不会用俯视的视角描述生活的失落，我看到的就是我们的父辈，面对失落时的艰辛探索。展示的就是自称社畜的你我，展示的是真实的中国。 就像豆瓣影评说的，我们期望一部好电影，期望一部值得尊敬的电影，一个值得尊敬的导演。谢谢贾樟柯。

读《人间佛陀释迦牟尼》有感

2023-04-28 ::
#book 
早些年对佛教的认知来源于电视剧《西游记》，一直认为佛祖只是存在与想象之中。通过这本书了解到原来佛陀是真实存在过的人物，名叫悉达多。主要介绍了他从出生到涅槃，从出家到觉悟成佛到传播佛法的历程。既然是真实人物，就有他的局限性，比如对比丘和比丘尼的戒律不一致，在今天看来都是不可理解的。整体读下来，感觉佛教离我们并不遥远，他其实也是我们每个人内心的思考，是一种生活哲学，只是通过戒律这种标现形式外化成佛教了。 本书共十六个章节，我想大致可以分为三个部分：悉达多诞生前的宗教氛围，悉达多的成长及觉悟，佛教的形成和传播。 “古代印度的宗教、哲学、逻辑、文学、艺术、语言学等都很发达”(P007)，也许他们不太擅长记录，留下来的可参考的文献不多，很多时候需要世界上其他地方的著作甚至作者提到玄奘的著作对于研究佛学的人说也相当重要。早在悉达多诞生前，就已经有好多种哲学传播，也有很多修行的人。 之前一直好奇，佛经都在写什么。就用书里一段佛陀母亲怀孕后，记录解梦人的描述感受下：“若母人梦见，白象入右肋，彼母所生子，三界无极尊”（P043）。之前在大足石刻看到有白象的雕刻但是不解其意，原来白象的形象有如此渊源。 悉达多出身优渥，是古印度北方十六个王国其中一个国王的孩子，来看看佛经中的描述：“又别简取三十二女，令助养育。以女八人，拟报太子；以女八人，洗浴太子；以女八人，令乳太子；以女八人，令其戏弄”(P068)，“造三时殿。一者暖殿，以拟隆冬。第二殿凉，拟与夏暑。其第三殿，用拟春秋二时寝息”。可即便是这样，悉达多也不快乐，他父亲饭净王总是担心他出家，所以想早早的就安排他的婚事，耶输陀罗最后成了他的王妃，没多久他们还生下了孩子。在一次出游的过程中，先后遇见了路边的老人，病人和死去的人，引起了悉达多的思考。后来从北门出游看见一个比丘之后，决定出家。最开始悉达多选择了苦行的修行方式，认为所有的痛苦都是由欲望造成的，而欲望又是从身体出发的，书中描写到：“为了断绝这一切，就要用苦行把身体的欲望降到最低，用苦行消耗这个身体的能量，直到精疲力尽(P109)。”，悉达多用这种方法修行了六年。最后悉达多得出结论：“我今行此最极之苦，而不能证出世胜智，即知苦行非菩提因，亦非知苦、断集、证灭、修道，必有余法当得断生老病死(P118)。”，从这之后，悉达多接受供养，包括食物和为其修建住所，既不提倡享乐也不提倡极度的苦行，这其实更符合人性和具有可行性。 关于之后的传播以及和质疑的观点进行辩论，甚至连其他的派别修行人全体皈依，很多有趣的故事就不重复了。佛经最早是通过记诵的方式口口相传，佛灭三四百年后才用文字记录下来，分装在三个篮子里，于是叫“三藏”，分别是《经藏》《律藏》《论藏》。我在想这和西游记中的“唐三藏“一名来源大概有些关系吧。 在作者的自序中也重点提到书名的由来：“佛教终究是历史文化背景下的产物，佛陀也是历史中存在过的真实人物。我所取的角度是文化，非仅宗教；是历史，非仅神话。”

读卡佛的小说和访谈录有感

2023-04-28 ::
#book 
最近看到哔哩哔哩一个up主(天真的和感伤的小说家)，一人分饰两角，模仿采访海明威的视频印象很深，说海明威接受采访的态度不好，和海明威一些个人化的故事，很有趣。有一天中午偶然间看到了《雷蒙德卡佛访谈录》的新书推广，就想着读一读，里面提到了他的代表作 《当我们谈论爱情时我们在谈论什么》，而且这个标题很吸引我，虽然读完了依然不知道我们谈论爱情时我们在谈论什么，在这之前其实我还不知道卡佛是谁~ 最开始不知道《当我们谈论爱情时我们在谈论什么》是一本小说集，看目录才知道书名只是这本小说集中的一个短篇。总共有十七篇短篇，只有不到三分之一的短篇我感觉比较清晰，其它的短篇读起来很费力，不像一般的故事有开头，发展，冲突，结束。让人摸不着头脑，不仅有开放式的结尾，还有开放式的开头。 在读了一部分《雷蒙德卡佛访谈录》后，作者提到了他创作这些小说的背景，才对刚才那本小说集了解多一点。记得很有意思的一个描述：由于卡佛常年吸烟，声音很小，好多人采访完回去整理磁带的时候，只剩下些细碎的环境音，卡佛自己说像是一根连着身体的香烟。卡佛结婚很早，他十八岁，妻子十六岁就有了孩子，她们不得不很早就要面对生活，两个人一边上班一边上学还要养孩子，这跟他创作的题材大多是诗和短篇小说很有关系，因为短篇小说创作周期更短，更容易拿到报酬。常年的压力使得卡佛后来很长一段时间酗酒，进了几次医院和戒酒机构以后，才摆脱了这种状态，他甚至说成功戒酒比他在写作上取得的成就更令他骄傲。有采访者问他对酗酒和写作的关系，他说完全没有关系。 其实我看这本访谈录的时候好像是在看另一本小说，作家本身的生活，经历也许是一种现实的小说吧，通过采访者和作家的对话里提到了卡佛对写作的看法，写作的技巧，和对文学的思考， 印象比较深的是卡佛认为小说和电影、戏剧等其它形式的艺术一样，没有什么不同，都是建立一种作品和读者的沟通，他强调小说要真诚，而且为了能让编辑注意到你的作品，不要把好的东西留到最后。

被讨厌的勇气，做自己的勇气

2023-04-28 ::
#book 
听说《被讨厌的勇气》这本书有一段时间了，但也没有想过一定要读一读，直到最近在处理一些情绪时，很害怕会被讨厌，自然的想起来了。在没有看内容之前仅看书名以为仅仅是在谈人际关系的问题，但读完后感觉大部分都是在讲自我成长，怎样从自我接纳的角度看待人际关系。当下我有很多困惑，也看到社会中大家的困惑，比如职场内卷，三十五岁焦虑，社会时钟给我们的压迫感，多多少少能通过青年与哲人的对话，找到些启发。本书以对话的方式展开，使得阅读起来比较轻松，也不一定所有的问题都有答案，但可以给我的探索提供一些角度。 现在比较流行原生家庭的讲法，认为是过去的创伤对我们现在的生活产生了很大的影响。但在本书中，哲人说“过去的经历对现在的影响并不是不存在，关键是经历本身不会决定什么，我们给过去的经历‘赋予了什么样的意义‘，这直接决定了我们的生活”。“无论之前的人生发生过什么，都对今后的人生如何度过没有影响”，“决定自己人生的是活在‘此时此刻’的你自己”。这很强调人的主观能动性，而不是陷入在过去的情绪里。 “为了满足别人的期望而活以及把自己的人生托付给别人，这是一种对自己撒谎也不断对周围人撒谎的生活方式”，这有可能是现在我们焦虑的根本原因，社会要求我们在什么样的年龄该工作，什么样的年龄该结婚，什么样的年龄该拥有什么，而我们好似也接受了这样的设定，往这个目标去努力，但仍然担心做不到，可能自己内心也有想要怎样过，但真的很需要勇气。 常常在想，自我接纳是什么呢，是承认自己是个普通人，接纳自己的平凡，这是很消极的想法吗？是在给自己找借口吗？我还是没有答案，来看看哲人怎么说的吧：“我们既不能丢弃也不能更换‘我’这个容器。没必要特别积极的肯定自己，不是自我肯定而是自我接纳。这两者有明显差异，自我肯定是明明做不到但还是暗示自己说‘我能行’或者‘我很强’，也可以说是一种容易导致优越情结的想法，是对自己撒谎的生活方式。自我接纳是指假如做不到就诚实的接受这个‘做不到的自己’，然后尽量朝着能够做到的方向去努力，不对自己撒谎”。“接受不能更换的事物，接受现实的‘这个我’，然后，关于那些可以改变的事情，拿出改变的‘勇气’，这就是自我接纳”。 “‘无法真心祝福过得幸福的他人’，那就是因为站在竞争的角度来考虑人际关系，把他人的幸福看作‘我的失败’，所以才无法给予祝福”。在当今职场竞争这么激烈的一个年代，这其实蛮正常的，“一旦从竞争的怪圈中解放出来，就再没有必要战胜任何人了，也就能够摆脱‘或许会输’的恐惧心理了，变得能够真心祝福他人的幸福并能够为他人的幸福做出积极的贡献”。那要怎样才能克服自卑感呢，哲人说“健全的自卑感不是来自与别人的比较，而是来自与‘理想的自己’的比较”。 “虽然不想被人讨厌，但即使被人讨厌也没有关系？‘不想被人讨厌’也许是我的课题，但‘是否讨厌我’却是别人的课题。即使有人不喜欢我，我也不能去干涉。”，哲人强调在处理人际关系时要学会课题分离，“基本上，一切人际关系矛盾都起因于对别人的课题妄加干涉或者自己的课题被别人妄加干涉。只要能够进行课题分离，人际关系就会发生巨大改变”，也给出了具体的方法：“辨别究竟是谁的课题的方法非常简单，只需要考虑一下‘某种选择所带来的结果最终要由谁来承担？‘”。 对我而言，本书提到的“课题分离”非常有实际意义，这让我想到自己的母亲，我想当我放弃干涉她的“课题”（其实就是她对我的干涉），不再幻想她能做出改变时，一直以来那根弦好像就松了似的，处理好我自己的课题足矣。
